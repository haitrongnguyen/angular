Pipes trong angular là những hàm đơn giản được sử dụng trong các biểu thức template để chấp nhận một giá trị đầu vào và trả về một giá trị đã chuyển đổi. Bạn có thể sử dụng pipes để chuyển đổi các chuỗi, số tiền, ngày tháng và dữ liệu khác để hiển thị trong template

1.Sử dụng LowerCasePipe và UpperCasePipe

Name Input Output
In lowercase: {{value}} {{value | lowercase}}
In uppercase: {{value}} {{value | uppercase}}

2.Sử dụng SlicePipe

Name Pipe Input Output
slice:0:4 {{str}} {{str | slice:0:4}}
slice:4:0 {{str}} {{str | slice:4:0}}
slice:-4 {{str}} {{str | slice:-4}}
slice with list data in array {{collection}} {{collection | slice:0:3}}

3.Sử dụng DecimalPipe

Name Pipe Input Code Output
Decimal {{pi}} {{"pi | number"}} {{pi | number}}
Decimal {{pi}} {{"pi | number:'4.1-5'"}} {{pi | number:'4.1-5'}}
Decimal {{pi}} {{"pi | number:'4.1-5'"}} {{pi | number:'4.15-20'}}
With digitsInfo and locale parameters specified: {{pi}} {{"pi | number:'4.1-5':'vi'"}} {{pi | number:'4.1-5':'vi-VN'}}

4.Sử dụng PercentPipe

Name Pipe Input Code Output
use percent: {{ a }} {{"a | percent"}} {{ a | percent }}
use percent: {{ b }} {{ "b | percent:'4.3-5'" }} {{ b | percent:'4.3-5' }}

5.Sử dụng CurrencyPipe

Name Pipe Input Code Output
use currency: {{ a }} {{"a | currency"}} {{a | currency}}
use currency: {{ a }} {{ "a | currency:'VND'" }} {{a | currency:'VND'}}
use currency: {{ a }} {{ "a | currency:'VND':'code'" }} {{a | currency:'VND':'code'}}
use currency: {{ b }} {{ "b | currency:'VND':'symbol':'4.2-2'" }} {{b | currency:'VND':'symbol':'4.2-2'}}
use currency: {{ b }} {{ "b | currency:'VND':'symbol':'4.2-2':'vi'" }} {{b | currency:'VND':'symbol':'4.2-2':'vi-VI'}}
use currency: {{ b }} {{ "b | currency:'VND'" }} {{b | currency:'VND'}}

6.Sử dụng DatePipe

Name Pipe Input Code Output
use date: {{ today }} {{"today | date"}} {{today | date}}
use date: {{ today }} {{ "today | date:'fullDate'" }} {{today | date:'fullDate'}}
use time : {{ today }} {{ "today | date:'h:mm a z'" }} {{today | date:'h:mm a z'}}
